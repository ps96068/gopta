Pentru UserActivity:

UserActivityStart - start sesiune cu device info

UserActivityResponse - durată calculată și formatată

UserActivityStats - metrici complete engagement

Pentru UserInteraction:

UserInteractionCreate - tracking individual
UserInteractionBulkCreate - tracking în batch
InteractionHeatmap - vizualizare pattern-uri orare
UserPath - analiza journey complet
InteractionFunnel - conversion funnel analysis
ProductInteractionStats - metrici per produs

Features importante:

Calcul automat durată sesiune (format human-readable)
Parse user agent pentru device info
Time since calculation ("acum", "5 minute", "2 ore")
Heatmap pentru identificare ore de vârf
Path analysis pentru optimizare UX
Funnel pentru conversion rate optimization
